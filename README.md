## Campus Errand Order Taking Service Platform

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/27474e51c650428c900077f5640cf37e/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Order Taker, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Announcement Information
6. Errand Tasks
7. Order Details
8. Delivery Address Management
9. User Complaints
10. Balance Change Records


**Order Taker Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Announcement Information
6. Errand Tasks
7. Order Details



**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Announcement Type Management
5. Task Type Management
6. Complaint Type Management
7. Order Details Management
8. Order Taker Management
9. Announcement Information Management
10. Errand Task Management
11. Delivery Address Management
12. User Management
13. User Complaint Management
14. Balance Change Record Management
